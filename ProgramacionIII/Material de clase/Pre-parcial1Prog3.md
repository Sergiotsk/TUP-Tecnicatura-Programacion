# Tecnicatura Universitaria en Programación
## Programación III

### Pre-Examen Parcial: Introducción a los lenguajes de Front-End HTML – CSS - JavaScript

---

## Bloque 1: HTML5 y Formularios (Teoría y Desarrollo)

**1. Desarrollo:** ¿Para qué sirve el atributo `name` en los elementos de un formulario y en qué se diferencia del atributo `id`?

**2. Código:** Escribe el código HTML para un formulario que contenga un input de email, un checkbox para "Aceptar términos" y un botón de envío.

**3. Opción Múltiple:** ¿Qué sucede si un formulario no tiene el atributo `method` especificado?
- a) Los datos se envían por POST de forma predeterminada.
- b) Los datos se envían por GET de forma predeterminada.
- c) El formulario no se envía y arroja un error de consola.

**4. Desarrollo:** Explica la importancia de la etiqueta `<label>` y cómo se vincula correctamente con un input para mejorar la accesibilidad.

---

## Bloque 2: CSS y Maquetación (Análisis y Código)

**5. Código:** Crea una regla CSS para un elemento HTML con clase `"parrafo_tipo1"` que modifique tanto la fuente a Arial, el color del texto a verde y el fondo a `#08A74B`, otra para todos los elementos `<h2>` que establezca el tamaño en 20px y por último una para un párrafo con `id="p1"` que muestre el texto en negrita.

**6. Opción Múltiple:** ¿Cuál es la diferencia entre `position: relative;` y `position: absolute;`?
- a) `relative` se mueve respecto a su posición original; `absolute` respecto al ancestro posicionado más cercano.
- b) `relative` saca al elemento del flujo del documento; `absolute` no.
- c) No hay diferencia si el contenedor padre es el `body`.

**7. Código:** Escribe las propiedades necesarias para centrar un `div` tanto horizontal como verticalmente dentro de un padre que tiene `display: flex;`.

**8. Opción Múltiple:** ¿Qué propiedad usarías para que los elementos hijos de un Flexbox se pasen a la siguiente línea si no entran en el ancho de la pantalla?
- a) `flex-direction: column;`
- b) `flex-wrap: wrap;`
- c) `justify-content: space-around;`

## Bloque 3: JavaScript y Manipulación del DOM (Lógica)

**9. Código:** Escribe **una línea** de JS que cambie el color de fondo de los párrafos hijos de un `div` con clase `"lista"` alternando color de fondo `"#FFFFFF"` para los pares y `"#A0AFAF"` para los impares.

**10. Desarrollo:** Describe el flujo de pasos que debe seguir un script para: 1. Leer un dato de un input, 2. Procesarlo y 3. Mostrarlo en un `<span>` vacío.

**11. Opción Múltiple:** ¿Cuál es la forma correcta de declarar una variable que no cambiará su valor a lo largo del programa?
- a) `var`
- b) `let`
- c) `const`

**12. Código:** Escribe una función llamada `saludar()` que tome el valor de un input (`id="usuario"`) y muestre una alerta que diga `"Hola [nombre del usuario]"`.

## Bloque 4: Eventos y Validación (Aplicación)

**13. Opción Múltiple:** Si queremos ejecutar una función apenas el usuario termina de escribir en un campo, ¿qué evento es el más adecuado?
- a) `onclick`
- b) `oninput`
- c) `onblur`

**14. Código:** Crea un listener (escuchador) en JS para que, al hacer clic en un botón con `id="btn-color"`, se cambie el color de fondo de la página (`document.body`) a amarillo.

**15. Desarrollo:** ¿Qué utilidad brinda el uso del atributo `disabled` en un campo de texto HTML?

**16. Código:** Escribe las propiedades necesarias para centrar un `div` tanto horizontal como verticalmente dentro de un padre que tiene `display: flex;`.
